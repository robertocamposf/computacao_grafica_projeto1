# Projeto 2D - Faculdade

Este projeto consiste em uma aplicação web com modelos 2D utilizando JavaScript.

## 🚀 Tecnologias
- HTML
- JavaScript
- CSS

## 🎮 Controles
- **Seta para Cima (↑):** Acelerar propulsor
- **Seta para Baixo (↓):** Frear / Marcha à ré
- **Setas Laterais (← / →):** Rotacionar a nave

## 📦 Como executar
1. Baixe o projeto
2. Abra o index.html no navegador
